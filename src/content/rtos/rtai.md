### Features

- RealTime Application Interface for Linux.
- Linux Real Time Module.
- Real Time Hardware Abstraction Layer.
- POSIX compatibility. Traditional RTOS IPCs including: Semaphores, mailboxes, FIFO, shared memory, and RPCs.
