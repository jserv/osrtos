# About OSRTOS

OSRTOS is a comprehensive resource for open source real-time operating systems (RTOS), embedded libraries, and projects. Our mission is to help developers discover, compare, and learn about RTOS solutions and related technologies for embedded systems development.

## What We Offer

- **RTOS Documentation**: Detailed information about popular open source real-time operating systems, including features, architectures, and supported platforms
- **Library Directory**: Curated collection of embedded libraries and middleware for various use cases
- **Project Showcase**: Examples of real-world embedded projects using different RTOS and libraries
- **Up-to-Date Information**: Automatically updated GitHub metadata including stars, versions, and release information
